# Helpdesk - Linguagem C e Estrutura de Dados

Programa de um HelpDesk simples escrito na linguagem C

**Duas versões principais:**

Usando fila, pilha e árvore binária:
```
    HelpDesk.c
```

Sem estrutura de daods:
```
    HelpDeskSEM.c
```
